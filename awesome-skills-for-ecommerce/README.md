## 📦 Skill Structure

Every skill in this directory follows this layout:

```
skill-name/
├── SKILL.md          # Main skill document (instructions for the AI agent)
├── README.md         # Human-readable overview
├── examples/         # Sample inputs & outputs
└── assets/           # Templates, images, configs
```
